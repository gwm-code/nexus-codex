# Nexus Codex Beta Program

## Eligibility
- Teams operating critical tooling with strong security requirements.
- Early adopters willing to provide feedback on orchestration workflows.

## Enrollment Steps
1. Review the user guide and architecture docs.
2. Run the onboarding checklist:
   - `cargo run -- cache-warm --root .`
   - `cargo run -- sandbox-test --command "cargo test" --root .`
   - `cargo run -- audit scan --root .`
3. Share findings and logs with the Nexus team.

## Feedback Channels
- Weekly checkpoints via shared reports.
- Optional dashboard access for telemetry review.
